#  Albedo Discord Theme

Tema personalizado para Discord inspirado en Albedo, personaje de Overlord.

Este proyecto fue creado con el objetivo de transformar completamente la apariencia de Discord utilizando CSS, logrando una estética oscura, elegante y consistente basada en tonos púrpura, transparencias y efectos visuales personalizados.

---

##  Características

- Interfaz completamente personalizada.
- Estética inspirada en Albedo y Nazarick.
- Fondo temático integrado.
- Transparencias y efectos glassmorphism.
- Colores púrpura personalizados.
- Efectos glow y resaltado visual.
- Mejor integración entre paneles, canales y mensajes.
- Diseño enfocado en una experiencia más inmersiva.

## 🎯 Objetivo del proyecto

Más que cambiar colores, el objetivo fue crear una identidad visual completa para Discord.

Quise que cada elemento de la interfaz siguiera una misma línea estética para generar una experiencia más inmersiva y visualmente atractiva, manteniendo la funcionalidad original de la aplicación.

---

## 🚀 Instalación

1. Descarga el archivo del tema.
2. Copia el archivo `.theme.css` en la carpeta de temas de BetterDiscord.
3. Activa el tema desde la sección de Themes.
4. Disfruta la nueva apariencia.
