Albedo God Theme Pack

Files:
- Albedo.theme.css
- assets/albedo-bg.png
- assets/reference-mockup.png
- CopilotPrompt.txt

Uso en BetterDiscord:
1. Copia `Albedo.theme.css` y la carpeta `assets` en tu carpeta de BetterDiscord themes.
2. Abre Discord, ve a Ajustes > BetterDiscord > Themes y activa `Albedo God Theme`.
3. Si no se ve el fondo, recarga Discord con `Ctrl+R`.

Notas:
- Esta versión utiliza la imagen `assets/albedo-bg.png` como fondo.
- Asegúrate de mantener los archivos CSS y la carpeta assets juntos.
- Para cambiar colores, abre `Albedo.theme.css` y edita las variables dentro de `:root`.
- El archivo debe terminar en `.theme.css` para que BetterDiscord lo detecte automáticamente.
